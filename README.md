# Resume Screening AI

An AI-powered resume screening system that matches resumes with job descriptions using semantic embeddings, TF-IDF fallback, NLP, and skill extraction.

## Features
- Resume parsing (.pdf, .docx)
- Semantic resume/job-description similarity scoring
- Skill extraction
- Match scoring
- Missing skill detection
- Candidate ranking
- Modern blue UI

## Run locally
```bash
python -m streamlit run app.py
